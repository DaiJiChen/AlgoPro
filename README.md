# AlgoPro: Algorithm Visualization, Explanation and Code Demo.
An native android app that visualize algorithms.

* Bubble Sort
* Selection Sort
* Insertion Sort
* Quick Sort
* Merge Sort
* Linear Search
* Binay Search
* BFS
* DFS
* Binary Search Tree
* Dijkstra's Algorithm
* HashMap
* Queue
